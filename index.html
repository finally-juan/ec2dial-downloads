Installation

After downloading:

  1. Make the file executable (Linux/macOS): chmod +x ec2dial.*
  2. Move to a location in your PATH: mv ec2dial.* /usr/local/bin/ec2dial

Usage

ec2dial [options] <hostname|instance-id>

For more information, run: ec2dial --help
